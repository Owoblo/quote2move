# AI Detection API Keys Configuration
# Add these to your .env file

# Option 1: OpenAI GPT-4 Vision (Recommended - Most Accurate)
REACT_APP_OPENAI_API_KEY=your_openai_api_key_here

# Option 2: Google Vision API (Alternative)
REACT_APP_GOOGLE_VISION_API_KEY=your_google_vision_api_key_here

# Instructions:
# 1. Choose ONE of the above services
# 2. Get your API key from the respective provider
# 3. Add it to your .env file
# 4. Restart the development server

# OpenAI Setup:
# - Go to https://platform.openai.com/api-keys
# - Create a new API key
# - Add it as REACT_APP_OPENAI_API_KEY

# Google Vision Setup:
# - Go to https://console.cloud.google.com/
# - Enable Vision API
# - Create credentials
# - Add it as REACT_APP_GOOGLE_VISION_API_KEY
